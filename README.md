# Medify

منصة كاتب طبي ذكي محيطي ثنائية اللغة مبنية وفق وثائق المشروع `DOC-01..19`.

## التشغيل

1. انسخ `.env.example` إلى `.env` وغيّر الأسرار.
2. شغّل `docker compose up -d --build`.
3. افتح `http://localhost:3100`.

حسابات العرض عند `DEMO_MODE=true`:

- أدمن: المنشأة `demo`، المستخدم `admin`، كلمة المرور `Admin123!`
- دكتور: المنشأة `demo`، المستخدم `doctor`، كلمة المرور `Doctor123!`

واجهة API موثقة على `/api/docs` من خلال تطبيق الويب، وفحص الصحة على `/api/health`.

الوضع التجريبي يوفر محاكاة آمنة للتفريغ والتلخيص والإرشاد والرفع. الربط بمزود STT/LLM ونظام المستشفى الفعلي يتطلب مفاتيح واتفاقيات معالجة بيانات، ولا تُحفظ المفاتيح في المستودع.
